This program converts MDF4 files into TeSSLatraces.

Usage: Mdf4TeSSLaConverter -i [Input path] -o [Output path]

The program is based on the free modules of https://www.turbolab.de/mdf_libf.htm and requires the file MDF4Reader.dll, which can be downloaded from there. The dll-file must be registered by regsvr32 MDF4Reader.dll.
